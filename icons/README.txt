Place your PNG icon files in this folder:

  icon-180.png        — 180×180px  (iOS home screen icon, apple-touch-icon)
  icon-192.png        — 192×192px  (Android home screen / PWA manifest)
  icon-512.png        — 512×512px  (PWA splash screen / high-res)
  icon-maskable-512.png — 512×512px (Android adaptive icon — keep main content in center 80%)

Design notes:
  - Background color: #00b4a6 (teal)
  - Recommended: 🧠 brain emoji or custom brain icon centered on teal background
  - Rounded corners are handled by the OS — use a square canvas
  - For maskable icon: keep the brain within the inner 80% safe zone

Tools to generate icons quickly:
  - https://maskable.app/editor  (maskable icon preview)
  - https://realfavicongenerator.net  (generates all sizes from one image)
  - https://pwa-asset-generator.netlify.app
